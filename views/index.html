I thought it might be worthwhile to describe my process through which I created this site. First, I designed the frontend in responsive HTML5, with some jQuery involved in the hide/view nav function on the mobile side. The stylesheets are coded in SASS. I used Hammer + Anvil for prototyping. Next, I created a simple Node.js application, which is used to serve the static pages I created. Finally, it's been deployed to Heroku via git, with DNS handled by Amazon Web Services' Route 53.
